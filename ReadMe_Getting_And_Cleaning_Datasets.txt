Download the source data from https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip and copy the UCI HAR Dataset folder to the RStudio working directory. Eg:(E:\Coursera)

The first part (#1. Merge Sets) merges the training and test sets. 

The second part (# 2. Extract only measures of mean and standard deviation for each measurement) uses the features file to create the mean and standard deviations

The third part (# 3. Uses descriptive activity names to name the activities in the data set.) to name the data set with descriptive varaible names

The fourth part (# 4. labels the data set with descriptive activity names.) labels the data set with descriptive variable names

The firth part (# 5. tidy data set with the average of each variable for each activity and each subject.) creates a secondary data set with average of each variable for each activity and each subject.
